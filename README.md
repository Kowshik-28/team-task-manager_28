# Team Task Manager Full Stack

This project contains:
- **React Frontend**: Powered by Vite and Axios.
- **Python Backend**: Built with Flask, SQLAlchemy, JWT, and bcrypt.
- **SQLite Database**: Local, zero-configuration database.

## Directory Structure
- `frontend/` - React application.
- `backend/` - Flask REST API application (with Python virtual environment).

## Getting Started

### Run Backend
1. Go to backend directory:
   ```bash
   cd backend
   ```
2. Start the Flask server:
   ```bash
   ./venv/bin/python src/app.py
   ```
   The backend will run on `http://127.0.0.1:5001`.

### Run Frontend
1. Go to frontend directory:
   ```bash
   cd frontend
   ```
2. Install npm dependencies (requires Node.js):
   ```bash
   npm install
   ```
3. Run Vite development server:
   ```bash
   npm run dev
   ```
